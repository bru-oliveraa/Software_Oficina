# Software_Oficina
## Projeto

Projeto de análise e modelagem de processos para uma oficina mecânica, propondo melhorias através da modelagem BPMN e da prototipação de um sistema de gestão.

## Documentação

### BPMN AS IS
Representa o processo atual da oficina, identificando atividades manuais, gargalos e oportunidades de melhoria.

### BPMN TO BE
Representa o processo otimizado com a utilização de um sistema informatizado para gestão da oficina.
